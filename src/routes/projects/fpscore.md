## Origins

FPS Core began with a desire to open-source the development done on [Project Isolation](/projects/projectisolation). In addition to the development of the game, I maintained comments throughout the code as well as documentation in the form of a github wiki, which described the functionality of all of the game's components.

This however proved to be difficult to maintain, and as I was starting to develop project-specific features that would not be applicable to a wider audience, I decided to fork the project, and, for ease of use, host it on the Unreal marketplace. The goals of the project were quite simple - **to provide an easy to use, simple framework to make developing FPS games easier**.

## Goals

To fulfil this goal, I refactored much of Project Isolation's codebase and stripped out unecessary features, focusing on the core elements of a first-person shooter (hence, FPS Core) to focus on a few key features:

- Movement, including advanced movements like sliding and vaulting
- Complete, easy to use weapon systems
- Data-driven modular weapons, including attachments
- Damage systems
- Interaction systems

I also decided to distribute it as a plugin, to make it easy to toss into new or existing projects, allowing anyone to use it, be they a new developer working on their first game, or an experienced team looking for a prototyping tool to build off of for their latest game.

I also wanted to provide suitable high-quality documentation, and so created a [Documentation Website](https://emmadocs.dev/) with the help of Wordpress on which I distribute guides and documentation on how to use the product, in collaboration with the discord server I'm able to provide high-quality support to all users.

## Results

Today, the plugin has over 45,000 downloads from the Unreal Engine marketplace, and has a rating of 4.69/5 (at the time of writing). I've been regularly updating it with bug fixes and new features since it launched in September of 2022.
