## Origins

FPS Core began with a desire to open-source the development done on [Project Isolation](/projects/projectisolation). In addition to the development of the game, I maintained comments throughout the code as well as documentation in the form of a github wiki, which described the functionality of all of the game's components.

This however proved to be difficult to maintain, and as I was starting to develop project-specific features that would not be applicable to a wider audience, I decided to fork the project, and, for ease of use, host it on the Unreal marketplace. The goals of the project were quite simple - **to provide an easy to use, simple framework to make developing FPS games easier**.
